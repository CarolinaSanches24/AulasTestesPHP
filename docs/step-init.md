#### Install composer 

### Initial Files

```php
composer dump
```
Este comando criará os seguintes arquivos no diretório raiz do seu projeto:

- `composer.json`
- `composer.lock`
- `vendor`
- `autoload.php`
- `autoload_namespaces.php`
- `autoload_psr-4.php`
- `autoload_classmap.php`
- `autoload_static.php`
